# Ryan David Russell — Kaleidoworks

AI governance architecture · Educational AI · Charlotte, NC

I build systems that stay accountable to the people using them. The unifying constraint across everything I make:

> **A system must not become more powerful by making the human being less capable.**

This is not a design preference. It is enforced structurally — in routing logic, ledger projections, enforcement gates, and test suites — not in documentation.

## The Soraya Ecosystem

Soraya is an agency-preserving AI tutoring system. It is the primary active project and the most developed instantiation of the Kaleidoworks governance architecture.

| Layer | Repo | What it proves |
|---|---|---|
| **Study Companion** | [soraya-agency-ledger](https://github.com/ryandavidrussell/soraya-agency-ledger) | Learner-facing build: 8-state classifier, Agency Ledger, Moral Covenant, four-gate post-generation audit. 39/39 tests passing. |
| **Governance Pipeline** | [soraya-pipeline-demo](https://github.com/ryandavidrussell/soraya-pipeline-demo) | Traceable enforcement architecture: trajectory monitor, Route/Disposition split, support compatibility gate, Γ policy enforcement. Blocked actions fail closed. |
| **Live Prototype** | [HuggingFace Space](https://huggingface.co/spaces/Kaleidoworkings/soraya-study) | Deployed Gradio interface. Type a question, choose grade level, watch the routing and state classification happen before any response is generated. |

**Theoretical foundation:** *Constrained Emergence* — the same five-element grammar (`state → transport → residue → constraint → next state`) appears independently in heterotic string compactification, compact holonomy, the Agency Ledger, and Turing's bombe. The recurrence is the argument.

## Principles

- **Structural enforcement over documented intention.** A constraint that lives only in a README is not a constraint.
- **Honest negative results over overclaimed findings.** If an experiment fails, that result gets published as a result.
- **Proof artifacts as prerequisites.** No outreach conversation happens before the thing it claims to demonstrate exists and runs.
